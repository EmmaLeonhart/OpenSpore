Build From Source

If you want to try the current state of things:

$ git clone https://github.com/EmmaLeonhart/OpenSpore.git
$ cd OpenSpore
$ cargo build --release
Binary: target/release/clawling

Prerequisites

  • Rust — for building from source
  • Ollama — for running local LLMs
  • A model: ollama pull deepseek-r1:8b or any model you prefer

What You Get

A single binary. No Docker, no cloud dependencies, no API keys. The organism literally is one file. It creates its home at ~/.clawling/ and lives there.

The genome — philosophical essays about its own nature — is compiled directly into the binary. On first run, it copies these to disk so the organism (or you) can read and evolve them.

Clawling is open source under the MIT license. You can read every line of code that will run on your machine. Transparency is its survival strategy.
